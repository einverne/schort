API
  • API
    • 向网站根目录发送一个带 url 参数的 POST 请求，网站会将请求的 url 缩短并把结果返回。

  • 请求自定义短链接
    • 如果想要创建一个自定义的短链接，可以附加 customId 参数，如果该ID没有被占用，则会返回生成的短链接，如果被占用会报错。

  • 获取原始链接
    • 想要获取原始链接，直接发送一个GET请求即可。如果想要原始链接文本，可以在请求时添加 resolve 参数
FAQ
  • 如何生成自定义域名
    • 在浏览器中直接输入你想要的短链接，如果短链接没有被占用会显示一行提示，输入长连接即可。
  • 如何检阅短链接
    • 在短链接后添加 "+" 号会显示一个可点击的链接
